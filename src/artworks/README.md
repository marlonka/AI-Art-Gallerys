# Inner AIye Artworks

Drop generated artwork images here.

Root files are shared replacements across the whole gallery:

```text
mona_lisa.webp
girl_with_pearl.webp
starry_night.webp
```

Room folders are scoped to a single room:

```text
room_1/mona_lisa.webp
room_2/great_wave_off_kanagawa.webp
room_3/birth_of_venus.webp
```

Use the existing artwork id or title as the filename when the image should replace a curated piece. Unknown filenames inside room folders are added as extra generated studies.
