# Security Policy

## Supported Versions

The `main` branch is the supported development line.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities. Use GitHub private vulnerability reporting:

https://github.com/marlonka/AI-Art-Gallerys/security/advisories/new

Include:

- affected file, route, or workflow
- reproduction steps
- expected impact
- any suggested fix or mitigation

Maintainers will review reports as soon as practical and coordinate fixes before public disclosure.

## Scope

Security-sensitive areas include build scripts, dependency updates, generated asset handling, local storage behavior, privacy/legal pages, and any future networked features.
